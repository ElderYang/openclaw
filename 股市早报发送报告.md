## 📋 配置修复说明

要成功发送消息到飞书，需要完成以下配置步骤：

### 方式一：Webhook URL（推荐）
1. 在飞书开放平台创建自定义机器人
2. 获取 Webhook URL（格式类似：https://open.feishu.cn/open-apis/bot/v2/hook/xxxx）
3. 将 URL 配置到 OpenClaw 的飞书渠道设置中

### 方式二：OAuth 授权
1. 在飞书开放平台创建企业自建应用
2. 配置应用权限（如：message.send）
3. 获取 App ID、App Secret、Tenant ID 等凭证
4. 在 OpenClaw 中配置这些凭证

### 配置位置（需确认）
通常在以下路径之一：
- `/Users/yangbowen/.openclaw/config/channels.json`
- `/Users/yangbowen/.openclaw/workspace/config/message_channels.yaml`
- OpenClaw Gateway 的配置界面
