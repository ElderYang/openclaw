# 5 步搭建自动化股票监控系统

家人们！今天分享一个超实用的自动化技能！😭

用 OpenClaw 搭建自动股票监控系统
每天自动推送持仓报告到飞书！

## ❶ 核心功能
- 每天 7:30 自动推送早盘分析
- 每天 17:30 自动推送收盘复盘
- 实时监控持仓个股价格
- 龙虎榜/资金流向自动追踪

## ❷ 配置步骤
1. 安装 OpenClaw（npm install -g openclaw）
2. 配置 API Keys（百炼/Tushare/QVeris）
3. 创建持仓列表（5 只个股 +3 只 ETF）
4. 设置飞书推送渠道
5. 配置定时任务（launchd）
