## 自动切换规则

### 触发关键词

**小红书助手**：
- "小红书"、"发布笔记"、"xhs"、"redbook"
- cron 任务：`AI 科技日报 `、`OpenClaw 实战 `、`AI 变现指南`

**股市分析师**：
- "股市"、"复盘"、"晨报"、"持仓"
- cron 任务：`morning_brief`、`afternoon_review`

### 切换流程

1. 检测任务/消息中的关键词
2. 读取对应 SOUL.md
3. 按照角色人设执行任务
4. 消息前缀添加角色标签（如 `【小红书助手】`）
