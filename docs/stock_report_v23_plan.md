## 📊 问题诊断

### 问题现象
v22 模板（`morning_report_template_v22.py`）大部分数据取不到，显示为空。

### 根本原因
**数据结构不匹配**：
- v22 模板期望的字段：`us_markets`, `zt_count`, `lb_count`, `overnight_news[].event/impact/sector`
- v21 主脚本提供的字段：`us_indices`, `limit_up{total_count, lianban_count}`, `overnight_news[].title/snippet`

### 为什么 v21 能取到数？
v21 的模板（`morning_report_template.py` 和 `afternoon_review_template.py`）使用的字段名和主脚本 `stock_review_v21.py` 提供的完全一致：
- `us_indices` ✅
- `limit_up` ✅
- `longhubang` ✅
- `holdings` ✅
- `indices` ✅
- `margin` ✅
- `industry` ✅
- `north_money` ✅
- `main_force` ✅

## 📋 v23 优化内容

### 早报 v23.0 优化点

1. **一句话导读**（30 字内，核心结论）
   ```
   外盘：美股普涨 A50+0.8% | 涨停 56 家连板 5 家
   ```

2. **隔夜大事**（3 条以内，事件→影响→板块）
   ```
   • 英伟达发布新芯片 → 影响 A 股 → 科技/AI
   ```

3. **隔夜外盘 + A 股影响点评**
   - 显示数据日期
   - 标注数据验证状态（✅/⚠️）
   - 直接给出 A 股影响判断

4. **持仓个股 + 美股对标**
   - 涨跌图标（🔺/🔻/➖）
   - 显示美股对标股票表现

5. **市场分析**（涨停 + 龙虎榜 + 资金）
   - 涨停股板块分布（v22 新增数据）
   - 知名游资动向（v22 新增数据）

6. **今日策略**
   - 仓位建议
   - 关注方向
   - 风险提示
   - 持仓策略

7. **昨日预测验证**（新增）
   - 自动加载昨日预测
   - 对比实际走势

### 复盘 v23.0 优化点

1. **一句话总结**
   ```
   A 股大涨 +1.5% | 涨停 56 家 | 龙虎榜 5 条
   ```

2. **涨跌分布可视化**
   - 🔺 红色上涨
   - 🔻 绿色下跌
   - ➖ 平盘

3. **涨停板深度分析**
   - 连板 Top5
   - 涨停股板块分布
   - 封板资金

4. **龙虎榜深度分析**
   - 净买入/卖出 Top5
   - 机构席位动向
   - 游资活跃股
   - 知名游资动向（v22 新增数据）

5. **市场主线**
   - 领涨板块 Top5
   - 龙头企业
   - 中军企业

6. **持仓个股表现**
   - 涨跌图标
   - 美股对标

7. **明日策略**
   - 分类型仓位建议（激进/稳健/保守）
   - 关注方向
   - 风险提示
   - 持仓策略
