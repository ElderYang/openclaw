# 步骤 1

## 安装 OpenClaw

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

✅ 等待 2 分钟完成
✅ 运行 openclaw --version 确认

---

# 步骤 2

## 配置小红书账号

1. 进入技能目录
2. 启动 MCP 服务器
3. 手机扫码登录
4. 检查状态✅

💡 登录一次管 364 天！

---

# 步骤 3

## 设置定时任务

```bash
openclaw cron add \
  --name "小红书日报" \
  --schedule "0 7 * * *"
```

⏰ 早 7 点自动发布
⏰ 午休/睡前时间更佳
