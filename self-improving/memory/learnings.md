## [2026-03-30] 自省系统增强

### 背景
原有自省系统过于简单，主要问题：
1. 日志扫描不够深入（只检查是否有 error 关键词）
2. 任务失败记录没有分析
3. 用户纠正没有深度分析
4. 反思报告流于形式

### 改进方向
1. **增强自省深度** - 扫描日志、分析失败、检查纠正
2. **事件触发反思** - 严重错误/重复问题/用户批评立即记录
3. **每周改进报告** - 问题分类、趋势分析、改进追踪

### 实施内容
- 修改 daily_self_reflection.py
- 创建 event_triggered_reflection.py
- 增强 weekly_reflection.py
