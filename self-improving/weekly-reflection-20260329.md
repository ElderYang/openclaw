## 💡 关键学习

### [LRN-20260325-001] 配置修改必须立即持久化
**类别**: 工作习惯  
**教训**: "mental notes"不生存，只有文件能持久化  
**新流程**:
```
修改配置 → 立即验证 → git commit → 安装配置 → 测试 → 记录 learnings
```

### [LRN-20260326-001] macOS Python 环境陷阱
**类别**: 技术配置  
**教训**: launchd 默认使用系统 Python，必须显式指定 Homebrew Python  
**配置模板**:
```xml
<key>ProgramArguments</key>
<array>
    <string>/opt/homebrew/bin/python3</string>
    <string>/path/to/script.py</string>
</array>
```

### [LRN-20260327-001] 缓存读取必须超时保护
**类别**: 性能优化  
**教训**: 文件读取也可能卡住（网络文件系统、权限问题等）  
**代码模式**:
```python
import signal
def timeout_handler(signum, frame):
    raise TimeoutError("读取超时")
signal.signal(signal.SIGALRM, timeout_handler)
signal.alarm(5)  # 5 秒超时
```
