## 允许误差范围

| 数据类型 | 允许误差 |
|---------|---------|
| 指数点位 | < 0.01 点 |
| 涨跌幅 | < 0.01% |
| 涨跌家数 | < 10 家 |
| 成交额 | < 1% |
