# 小红书运营记忆库

## 📕 爆款方法论

### 标题公式
- 15-20 字 + emoji
- 悬念/痛点/数字
- 例："AI 圈炸了！GTC 2026 黄仁勋发布了这些🔥"

### 内容结构
```
开头：口语化问候 + 身份引入 + 痛点共鸣
正文：3-5 个要点，每点 1-2 句
结尾：互动引导 + 行动号召
```

### 规避 AI 检测
- ✅ 口语化表达
- ✅ 个人感受
- ✅ emoji 自然点缀
- ✅ 段落长短变化
- ❌ 机械化列表
- ❌ 过于正式
