### [2026-03-26 14:24] Python 环境配置问题
**教训**: 
- macOS 有两个 Python 环境：系统 Python 和 Homebrew Python
- launchd 定时任务默认使用 `/usr/bin/python3`
- 解决方案：脚本中明确指定 `/opt/homebrew/bin/python3`
- 或者使用 `pyenv` 统一管理

### 系统化建设经验
1. **自动化优先** - 能自动的不要手动
2. **持久化配置** - 所有配置写入文件，不要依赖记忆
3. **分段输出** - 长回复必须分段，避免截断
4. **角色标签** - 硬编码到脚本，不要动态判断
5. **主动汇报** - 任务进度主动告知，不要等用户问
